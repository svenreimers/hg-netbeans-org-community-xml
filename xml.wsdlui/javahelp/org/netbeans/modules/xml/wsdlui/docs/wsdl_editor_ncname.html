NCNAME Data Type

See Also

XML Schema includes a data type called NCName. Any name that has the NCName data type must comply with the following rules:

  • The name must begin with a letter or an underscore (_).
  • The valid characters for the remainder of the name are:
    • Letters
    • Digits
    • Period (.)
    • Hyphen (-)
    • Underscore (_)
